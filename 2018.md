1. 判断对象中元素是否存在？
```JavaScript
const obj = {
  x: 'temp'
};

// 属性是否是对象的自有属性
function isOwnProperty(obj, key){
  return obj.hasOwnProperty(key);
}

// 是否是对象的属性 
function isProperty(obj, key){
  return typeof obj.key;
}
```
